# Usage

To add edge, vertice and its corresponding weights of the graph. <br/>

```cpp
GraphStruct graph(V, E);
graph.edge[i].src = 0;
graph.edge[i].dest = 1;
graph.edge[i].weight = 10;
```

To run minimum spanning tree, <br/>

```cpp
minimum_spanning_tree m;
m.run(graph);
```
